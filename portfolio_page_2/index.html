Games

Nodes

There are 100 nodes rendered onto the screen and if a node is within 100 pixels from another node a line is drawn between them. By having this basic rule, the program produces distinct patterns.

Matrix

This is a program I made for my generative media class. The program outputs random words from the code it was written with onto the screen.

Space Tone

In this program each object animated on the screen revolves in proportion to one of the eight planets and pluto. If an object crosses a particular point, it will play a distinct note provided by the Tone JS library.

Pong

I recreated the classic video game pong using JavaScript. I built an AI system that plays against a human player along with a scoreboard that keeps track of both players' scores.

Breakout

This is a game called Breakout that I recreated using JavaScript. The aim is to hit the ball with the paddle and remove the grid of bricks that are rendered on the screen.
