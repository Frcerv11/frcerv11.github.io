Arduino

Asteroids

I recreated the classic arcade game Asteroids using Processing and the Arduino UNO. To control and move the space ship left and right, I used a potentiometer. To shoot missiles, the user presses a button.

RC Car

This is an RC Car built with an Arduino UNO. I downloaded a driver that allowed me to control it with an XBOX remote. To program this robot, I used the JavaScript library Johnny-Five JS.
